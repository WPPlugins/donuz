Integra a plataforma donuz com o Woocommerce.

== Description ==

Integra a plataforma donuz com o Woocommerce para pontuar de forma automatica os clientes participantes de seu programa de fidelidade que efetuarem compra em seu Woocomerce.

== Installation ==

* Envie os arquivos do plugin para a pasta wp-content/plugins ou usando o instalador de plugins do WordPress.
* Ative o plugin.
* No menu Woocommerce - Donuz, Insira as informações fornecida pela Donuz.
